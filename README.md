# Roo & Kanga

This repository contains **configuration files, mechanical design assets, and documentation** for the **Roo** and **Kanga** platforms.

---

## Folder Overview

### `roo/`

Contains files related to the **Roo** platform.

* **`3d_files/`**
  Mechanical design files used for fabrication and prototyping.

* **`params/`**
  Parameter files used to **store and load parameters on the Pixhawk 6C**.

* **PDF files**
  Documentation, system diagrams, and control architecture references.

---

### `kanga/`

Contains files related to the **Kanga** platform.

* **`3d_files/`**
  Mechanical design files specific to Kanga.

* **`params/`**
  Parameter files used to **store and load parameters on the Pixhawk 6C**.

---

## Parameters

The `.params` files in this repository are used to **store configuration parameters for the Pixhawk 6C**.
These files can be loaded onto the controller to configure system behavior, hardware settings, and tuning values.
